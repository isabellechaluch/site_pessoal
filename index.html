Habilidades

HTML5

É uma linguagem de marcação utilizada para a criação e estruturação de páginas web, permitindo a criação de documentos que podem ser interpretados pelos navegadores. É a base fundamental para o desenvolvimento de sites e aplicações web, permitindo a organização e formatação de conteúdo.

CSS

Cascading Style Sheets, é uma linguagem de estilo utilizada em conjunto com o HTML para definir a aparência e o layout de uma página web. Em conjunto com o HTML, O CSS permite controlar elementos como cores, fontes, tamanhos, espaçamentos e posicionamento dos elementos na página.

Bootstrap

É um framework front-end popular e amplamente utilizado para desenvolvimento web. Ele oferece um conjunto abrangente de ferramentas e componentes que permitem criar sites responsivos e visualmente atraentes. Com sua grade flexível, componentes pré-fabricados e recursos de personalização, o Bootstrap agiliza o processo de desenvolvimento.

Projeto de pesquisa SENAI

Ao longo do meu curso técnico em biotecnologia no SENAI, desenvolvi um projeto de pesquisa sobre análises microbiológicas em busca de coliformes fecais em uma praia de Santa Catarina. Com base em métodos e técnicas apropriadas, realizei experimentos detalhados para coletar e analisar amostras de água da praia. Meu trabalho resultou em uma avaliação da qualidade microbiológica da água e das condições sanitárias da praia, com destaque para a presença de coliformes fecais. Recebi a nota 9,7 pela minha pesquisa e estou disponibilizando meu trabalho escrito no link abaixo.
