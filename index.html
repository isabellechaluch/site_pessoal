Linguagens de programação

HTML5

É uma linguagem de marcação utilizada para a criação e estruturação de páginas web. É a base fundamental para o desenvolvimento de sites e aplicações web, permitindo a organização e formatação de conteúdo de maneira hierárquica.

CSS

Cascading Style Sheets, é uma linguagem de estilo utilizada em conjunto com o HTML para definir a aparência e o layout de uma página web. Enquanto o HTML é responsável pela estrutura e organização do conteúdo, o CSS permite controlar elementos como cores, fontes, tamanhos, espaçamentos e posicionamento dos elementos na página.
