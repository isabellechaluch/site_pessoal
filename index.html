Através da programação, posso dar vida às minhas ideias mais criativas e resolver problemas complexos. Cada desafio é uma oportunidade para aprender e evoluir, e a sensação de ver meu trabalho ganhar vida é simplesmente gratificante.

Linguagens conhecidas

HTML5

É uma linguagem de marcação utilizada para a criação e estruturação de páginas web, permitindo a criação de documentos que podem ser interpretados pelos navegadores. É a base fundamental para o desenvolvimento de sites e aplicações web, permitindo a organização e formatação de conteúdo.

CSS

Cascading Style Sheets, é uma linguagem de estilo utilizada em conjunto com o HTML para definir a aparência e o layout de uma página web. Em conjunt com o HTML, O CSS permite controlar elementos como cores, fontes, tamanhos, espaçamentos e posicionamento dos elementos na página.
